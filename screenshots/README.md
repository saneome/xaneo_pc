# Скриншоты Xaneo PC

Эта папка предназначена для хранения скриншотов приложения Xaneo PC.

## Требуемые скриншоты

Для полного README.md необходимы следующие скриншоты:

1. **onboarding.png** - Экран онбординга с 3D эффектами
2. **login.png** - Экран входа с эффектом стекла
3. **register.png** - Экран регистрации с формой
4. **settings.png** - Модальное окно настроек
5. **about.png** - Окно "О приложении"

## Рекомендации по созданию скриншотов

1. Запустите приложение в release режиме:
   ```bash
   flutter run --release
   ```

2. Сделайте скриншоты в следующих состояниях:
   - Онбординг: все три этапа (можно сделать скриншот второго этапа)
   - Экран входа: с заполненными полями
   - Экран регистрации: с открытой формой
   - Настройки: открытое модальное окно
   - О приложении: открытое информационное окно

3. Сохраните скриншоты в формате PNG с разрешением не менее 1280x720

4. Назовите файлы согласно списку выше

## Альтернативный вариант

Если у вас нет реальных скриншотов, можно использовать placeholder изображения или создать их с помощью графического редактора, имитируя интерфейс приложения.
